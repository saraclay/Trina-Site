ITP 140 Mobile Application Technologies

App Assets

For your mobile app, you may need various assets such as audio, icon, and image files.

  • Audio
  • Icons
  • Image Tools

Audio

  • Android ringtones, notification & UI sounds and alarms
  • dev_tones: professional royalty-free sounds
  • Octave: A free library of UI Sounds
  • SoundKit

Icons

You may will need to create an app icon. You can use image tools to create them. You may also need other images such as ones for the tab bar or navigation bar.

  • App Icon Template
  • App-Bits
  • Glyphish
  • Icon Beast
  • Icons 8
  • Pixeden
  • The Noun Project

Image Tools

Using an image tool, you can create an image for each screen of your app.

Adobe Illustrator

Adobe Illustrator graphic design software is the industry's premier vector-drawing environment for creating graphics that scale across media.

Adobe Photoshop

Adobe Photoshop is a great tool for creating images. With its layering ability, you can change text and hide/show various layers. There are some great free templates available.

Pencil

Pencil is an open-source GUI prototyping tool that's available for ALL platforms. It is built for the purpose of providing a free and open-source GUI prototyping tool that people can easily install and use to create mockups in popular desktop platforms.

Sketch

Sketch gives you the power, flexibility and speed you always wanted in a lightweight and easy-to-use package. Finally you can focus on what you do best: Design. Use it to design interfaces, websites, and icons. It's available for the Mac only.
