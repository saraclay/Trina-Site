Your Web Space

Physics Libraries

Students can publish personal pages on a USC web server which we are going to use for this class. You will create a main page in which to list all of your completed assignments AND have hyperlinks to them.

File Transfer Program

In order to do this, you will need to use an FTP (file transfer program) program to copy the completed Web pages AND files (e.g. images) from your computer to the Web (the USC server where your student Web space is). On the laptops in class, FileZilla (Start → Programs → Internet Tools) is on the Windows machines and Cyberduck (Finder → Applications) in on the Mac machines. USC offers FileZilla (Windows) and Fetch (Mac) as the free FTP programs. You can download the free university distributed software at software.usc.edu.
Web Space
  1. Start FileZilla by selecting Start → Programs → Internet Tools → FileZilla.
  2. Click on the Connect image in the upper left corner. Do not use QuickConnect.
  3. In the "My FTP Sites" folder listing, click the "New Site" button and create a site called "aludra".
  4. Fill in the following information:
  • For the Host, enter aludra.usc.edu.
  • For the Servertype, select SFTP using SSH2. The Port should change to 22.
  • For the Logintype, select Ask for password.
  • For the User, enter your USC username.
  • Click the Connect button to log in. You will be prompted to enter your password.
Once you are logged in, you should see two "directory" panes. The one on the left represents the computer you are physically sitting in front of. The one on the right represents the USC Server named aludra which houses your web space.
  • To create a folder, right click on the mouse and select the "Create Directory" option.
  • To transfer files, double click on a file in one of the panes. It will then appear in the other pane. You can transfer files to aludra and also get files from aludra.

Fetch

  1. Start Fetch by using Finder and selecting Applications → Fetch. You can also use the Spotlight tool and enter Fetch.
  2. In the "New Connection" window, enter:
    • For the Hostname, enter aludra.usc.edu.
    • For the Username, enter your USC username.
    • For the "Connect using" option, select SFTP.
    • For the Password, enter your password.
    • Click the Connect button.
    • One you are logged in, you will see window representing the web server aludra.
  • To create a folder, click the "New Folder" icon at the top of the window.
  • To transfer files, just drag and drop files from your computer (from your Desktop or using a Finder window). You can also drag and drop files from aludra to your computer.

Cyberduck

  1. Start Cyberduck by using Finder and selecting Applications → Cyberduck. You can also use the Spotlight tool and enter Cyberduck.
  2. Click the "Open Connection" icon.
  3. Fill in the following information:
    • For the first pulldown, select SFTP (SSH File Transfer Protocol) instead of FTP (File Transfer Protocol). The Port should change to 22.
    • For the Server, enter aludra.usc.edu.
    • For the Username, enter your USC username.
    • For the Password, enter your password.
    • Click the Connect button.
    • One you are logged in, you will see window representing the web server aludra.
  • To create a folder, click the "Action" icon at the top of the window and select "New Folder ...".
  • To transfer files, just drag and drop files from your computer (from your Desktop or using a Finder window). You can also drag and drop files from aludra to your computer.

Setting up your web space

  1. First, run an FTP client to connect to aludra.
  2. All of your files that you want the world to see via a web browser needs to be in a folder called public_html. If it does not exist, create a folder and rename it to public_html.
  3. Open up the public_html directory by double clicking on it.
  4. For this class, all of your files should be be in a folder called itp140. If it does not already exist, create a folder named itp140. Please note that folder and file names ARE case-sensitive (i.e. itp140 is NOT the same as ITP140).
  5. Open up the itp140 directory by double clicking on it.

Access your web space

Using a web browser such as Firefox or Safari, enter the following web address: http://scf.usc.edu/~username/itp140.
