ITP 140 Mobile Application Technologies

Study Guide - Review

Review the Intro to Mobile slides for Green class for an introduction to mobile devices. Know the different operating systems and be able to match a device with an operating system. Know which mobile operating system has the most market share and which one has the most app revenue.

Review the Internet slides for the history of the Internet (know the role that USC played), the difference between the Internet and the WWW, and domain names. Be able to give examples of gTLD and ccTLD. Know what makes the Internet work. Be able to select the appropriate decade for the start of the Internet and the WWW. Understand what a killer app is and be able to give an example.

Review the App Types slides to know what mobile refers to and the difference types of mobile apps. What are some pros and cons for native apps, cross-compiled (hybrid) app, and web apps? Know what responsive web design is.

Review the Web slides to know what the acronyms HTML and CSS mean and the 2 components of HTML pages.

Review the Mobile Design slides to know Fitts's Law, Josh Clark's Rule of Thumb, the desired hit target, and the 4 steps in designing a mobile app. Know some of the mobile design patterns and be able to give examples.

Review the Apps slides to know the difference app styles. Be able to give an example of each. Know the difference between user experience, user-centered design, user interface, and interaction design.

Review the Colors, Images, and Icons slides for the image terminology. Know the different image formats we use on web pages and mobile devices. Understand the different color modes and model. Know the dos and don'ts of icon images.

Review the Wireframes slides to understand what a wireframe is and its main purpose.

Review the Interface Design slides to know the eight golden rules.

Review the Monetization slides to know the different ways to made money with apps and who pays.

Review the iOS User Interface slides and Android User Interface slides to be able to recognize the various UI components. Understand the three types of UI controls.
